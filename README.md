# LLM-Talmud-Translation
This repository contains the notebooks, intermediate artifacts, and sample results used to evaluate LLM translations of Tractate Sotah, with an emphasis on detecting potential “smoothing” of harsh or morally challenging content.
#Download Datasets
Use the "Download Talmud Data" to download the original Hebrew and English translations of all tractates from Sefaria (The Full Talmud- GPT 3.5 folder contains the relevant dataset) 
Use the "Download Sotah  Data" to download the original Hebrew and English translations for the selected sections of Sotah used in the paper. (the file "sotah_passages_with_text.json includes the relevant Sotah content, it is in the folder Sotah Analysis Code.)
#Perform Analysis 
The code for running teh analysis for an individual tractate or section can be found in "Sotah Analysis Code" for each of the models tested.
The code for running the entire Talmud is in "Batch Translation Code." 
#Results
Evaluator comments can be found in teh Sotah 3.5 and 5.2 comparison file, as well as the QA assessment file.
The ouput files can be found in teh QA Results folder and the Translation_Results folder
